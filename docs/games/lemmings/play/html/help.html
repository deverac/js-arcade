Lemmings™ - Help

The Game

Lemmings™ is an intriguing game in which you help hordes of otherwise-mindless creatures - knows as walkers - escape hostile environments.

Lemmings™ drop through a trapdoor onto each screen and, as they're creatures of very little brain, they need all the help they can get to survive the dangers that lie within.

To assist these cuddly, yet suicidally stupid creatures you use your mouse (similar to a lemming but it has buttons) to select a skill from the panel of icons at the base of the screen, to perform that skill. You have a strict time limit in which to get a required percentage of the little perishers through each level.

A quick brain, the ability to plan ahead and lifetime's dedication to save a Lemming Campaign are required to get the right Lemming to perform the right action at the right time. A forgotten Blocker or a misplaced bridge could spell disaster for every lemming on screen.

The Lemmings

A lemming can do the following:

  • Climb
  • Float
  • Blow Up Blockers (he he)
  • Block
  • Build Bridges
  • Dig Tunnels - across, diagonally and down

Lemmings™ are named after their skill; when a Lemming has a climbing skill he is called a 'Climber', when he is building a bridge he is called a 'Builder' and so on.

Points to note

Builders stop constructing bridges:

  • When they run out of blocks
  • If the bridge hits a solid object
  • If the Lemming hits his head untill they fall

Miners and diggers keep digging untill they fall.

Bashers keep digging until there is no more land in front of them.

Metal (shown as square plates and usually used to contain liquids) can not be dug through or exploded away.

A Climber or Floater retain these skills untill the current level is won (or lost) or untill they die - a Lemming given both these skills becomes an Athlete.

Starting a game

Lemmings™ is organised into four increasingly difficult game categories: FUN, TRICKY, TAXING and MAYHEM. The FUN category consists of 30 levels and is designed to familiarise you with the game concept and how to work with Lemmings™ skill attributes. You don't have to start with this category: You can go directly to any of the higher skill categories but we suggest that you begin with at least a few of the levels in the FUN category to get the feel of the game before proceeding to the more difficult categories.

TRICKY, TAXING and MAYHEM increase in difficulty with the MAYHEM level being the most difficult. Each of the game categories has 30 levels*

To start a game of lemmings™, just select the desired difficulty from the main menu. You will be presented a selection box with the levels available for that difficulty. When you haven't played a level yet the first level will be highlighted, and the code for that level will be already filled in. Just press 'Play!' to let the action begin!

* Note that because this is just work in progress not all levels and difficulties are available yet.

Passwords

Each level in Lemmings™ requires a password to be played. You only receive the password for a next level once you have completed a level. This password will automatically be added to the game selection screen, on which also is indicated which levels you already completed.

The original game accepted multiple passwords for a single level; this version has fixed passwords per level, so a passwords found on the internet for a specific level may not work in this game!

The passwords and the levels completed are stored in cookies; if you don't accept cookies, or cleaned up your cookies you will have to start from scratch the next time you play DHTML Lemmings™.

Controls

Lemmings™ is normally played with the mouse. You can select a skill from one of the icons at the bottom of the screen by clicking on it (the button becomes animated) and than pointing and clicking on a lemming. You can see which lemming will be selected by the target area surrounding the lemming when you hover over it with your mouse. The statusbar on the bottom-left will show you the current skill of the selected lemming and it's number (lemmings™ don't have names).
If you press the right or left arrow on your keyboard, you will only target lemmings™ that go right or left at that moment.

Not all skills are available in every level; the number in the icon will tell you how many of each skill you can use. If it doesn't show a number than that skill is unavailable in that particular level.

All icons will show you it's purpose when you hover over it.

Furthermore notice the scrollbar; many levels extend beyond the visible part of the screen. If you cannot find the exit, try scrolling left or right.

Icons Explained

From left to right these are the icons and their alternative keys:

  • Climber - Climbs vertical surfaces ('R')
  • Floater - A brolly opens to ensure a safe descent ('S')
  • Exploder - Bomb to dispatch single Lemmings™ (one way to get rid of Blockers) ('T')
  • Blocker - stands with arms outstretched to block the passage of fellow Lemmings™ ('U')
  • Builder - builds bridges, each limited to twelve steps. When a Lemming has laid his twelfth brick he turns to look at you for a moment... if you click on him again he will continue building. ('V')
  • Basher - digs horizontally, will only dig when a suitible surface is directly ahead of him. ('W')
  • Miner - uses a pick to dig diagonally down ('X')
  • Digger - burrows vertically ('Y')
  • Increase flow of Lemmings™ on to the level ('P')
  • Decrease flow of Lemmings™ on to the level up to the displayed minimum ('Q')
  • Pause - gives you time to think ('Z')
  • Fast Forward - once the road has been laid out by your scouts you can use this button to quickly advance all the remaining lemmings™ to the exit. Note that the time will run faster too!
  • Nuke all lemmings™ - should you find yourself in a 'no-win' situation (double-press to activate) ('{')

Other Controls

On the very bottom of the screen, next to the statusbar are a couple more fields that give you information during the game:
'Out' tells you how many lemmings™ are currently walking about the level
'Home' tells you how many lemmings™ have safely made it to the exit
'Time' tells you how many time you still have left in the game

Furthermore the following keys can be used to control sound and animations during play:

1 - turns music and sound off during the level
2 - turns music and sound on during the level
3 - turns animations off during the level
4 - turns animations on during the level

Don't use you numpad for these keys; that won't work.
Note that if you want to have these options applied to all levels, you need to set the options in the main menu.

Last but not least you can use the '/' key to immediately quit a level and return to the main menu.

Requirements

Unlike the original game DHTML lemmings™ requires a lot of resources (around 10 - 20MB of memory) and a fast processor to run smoothly. A 500MHz processor will do, but above 1GHz it becomes much more enjoyable. You will need at least a 2GHz processor for the fast forward mode to work correctely (AMD's are faster than P4's :P )

Javascript isn't a fast programming language because it needs to be interpreted at execution time. However it does have the advantage that most browsers are javascript enabled, so you won't need anything more than a modern browser.

DHTML Lemmings™ has been written to work in IE5.5 and above and in recent Mozilla versions (around 1.3 and higher). Other browsers (such as Opera) may or may not be able to run DHTML Lemmings™
